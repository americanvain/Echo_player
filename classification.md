# 问题识别器 — 精简总结（基于原文）


---

1. 语音 / 音系层
- 在干什么：听清音素；处理连读、弱读、吞音；识别词边界。
- 典型表现：把一个声音听成另一个词；听到“熟悉但不该出现的词”；句子从第一个词就开始崩。（典型“英语听力差”）

2. 词形识别层
- 在干什么：把声音映射到“词条”；在多个候选词中选对那个。
- 典型表现：明明听到了声音，但激活了错误的词；正确词你“没想到”；一看字幕就恍然大悟。（输入经验不足）

3. 词义层
- 在干什么：为词条激活正确义项；抑制不合适的义项。
- 典型表现：不认识单词；熟词生义理解错；只知道一个意思，用错语境。（最常见、易被误判为“英语不好”）

4. 句法解析层
- 在干什么：分配主谓宾；解析修饰关系；处理从句、倒装、省略。
- 典型表现：谁对谁做了什么搞乱；听到一半突然“脑子走歪”；长句一来就崩。（句法能力不足）

5. 组合语义层
- 在干什么：把词义 + 句法 → 事件意义；构建“发生了什么”。
- 典型表现：每个词都懂；结构也看得懂；但整体意思不对 / 很怪。（中高级阶段常见问题）
